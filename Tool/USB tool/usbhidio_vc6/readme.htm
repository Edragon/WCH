Usbhidioc

This is the readme file for the usbhidioc application, which includes host software for communicating with USB HID-class devices.

Usbhidioc is brought to you by

Jan Axelson
Lakeview Research
http://www.Lvr.com

(Usbhidioc means Universal Serial Bus (usb) Human Interface Device (hid) Input/Ouput (io) application in C (c).

Files

In additon to this file (readme.htm), usbhidioc includes various files created by the App Wizard in Visual C++ 6. The project's readme.txt file has details.

The project requires system files included in the Windows DDK. For more information about obtaining these files, see my web site's USB page (www.Lvr.com/usb.htm). The files include hid.h, hidsdi.h, setupapi.lib, and setupapi.h. You may have to set the path for these in Project > Settings > Link > Input.

More about the Code

This code has been tested under Windows 98, Windows 2000, and Windows XP.

The project will also load into and run under Visual Studio.

For device firmware that works with this application , see my HID Page at www.Lvr.com/hidpage.htm.
